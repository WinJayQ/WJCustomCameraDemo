# WJCustomCameraDemo
自定义相机界面，支持多张拍照。（下一张，重拍，完成）

##介绍
###1)自定义相机界面，取消，拍照，本地相册，重拍，下一张，完成
###2)支持多张拍照
###3)拍照完成后可以预览所拍照片
